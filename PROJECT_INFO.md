# Snake Game Web App

## Project Information

**Name:** Snake Game Web App
**Version:** 1.0.0
**Description:** An immersive snake game with stunning visual effects and sound design.

## Technologies Used

- HTML5 Canvas for rendering
- CSS3 animations and gradients
- JavaScript ES6+ for game logic
- Web Audio API for sound effects
- Responsive web design

## Features

- Classic snake gameplay with modern UI
- Immersive sound effects for all game events
- Beautiful visual effects including glowing elements
- Particle background effects
- Responsive design for all devices
- High score tracking using localStorage
- Increasing difficulty as score increases

## How to Run

1. Clone the repository
2. Open `index.html` in a web browser
3. Use arrow keys or WASD to control the snake
4. Press SPACE to pause/resume
