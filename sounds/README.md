# Sounds Directory

This directory contains all sound files used in the Snake Game webapp.

## Sound Files

- `eat.mp3` - Sound effect played when snake eats food
- `gameover.mp3` - Sound effect played when game is over
- `move.mp3` - Sound effect played with each snake movement
- `background.mp3` - Background music that loops during gameplay

Note: These are placeholder files. In a production environment, you would replace these with actual sound assets.
